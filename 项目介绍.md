# 离散数学智能教学体 - 项目介绍文档

## 一、项目概述

**离散数学智能教学体**是一个基于 Web 的离散数学智能化教学辅助平台，旨在为高校离散数学课程提供全方位的教学支持。系统整合了知识管理、智能题库、课件制作、图形可视化、AI 助教等核心功能，采用**青绿风**传统中式美学设计风格，打造沉浸式的学习体验。

### 核心价值

- **知识结构化**：将《离散数学及其应用》（Rosen）教材内容系统化整理为可检索的知识条目
- **智能学习路径**：基于知识图谱和学习数据分析，推荐个性化学习方案
- **互动式练习**：支持章节练习、随机刷题、模拟考试等多种练习模式
- **可视化教学**：内置图形画板，支持图论、树等离散数学概念的可视化绘制
- **AI 辅助教学**：集成多平台 AI 能力，提供智能答疑和个性化辅导

## 四、数据模型

### 4.1 核心模型关系

```
User ───┬──→ Conversation ───→ Message
        │
        ├──→ QuizAttempt ───→ QuizAnswer
        │
        └──→ WrongQuestion ←── Quiz
                 │
KnowledgeItem ←──┼──→ Example
                 │
                 └──→ Courseware ←── Graph
```

### 4.2 模型清单

| 模型 | 表名 | 核心字段 | 说明 |
|------|------|----------|------|
| User | users | id, username, password_hash, is_admin, api_key | 用户信息 |
| KnowledgeItem | knowledge_items | id, title, content, chapter, tags | 知识条目 |
| Quiz | quizzes | id, question_text, answer, question_type, difficulty, tags | 试题 |
| Courseware | coursewares | id, title, mode, content_json | 课件 |
| Graph | graphs | id, name, type, data_json | 图形数据 |
| ExamPaper | exam_papers | id, title, questions_json, pdf_path | 试卷 |
| PPTSlide | ppt_slides | id, chapter, title, content, group_id | PPT 幻灯片 |
| Conversation | conversations | id, user_id, title | 对话会话 |
| Message | messages | id, conversation_id, role, content, sources | 对话消息 |
| QuizAttempt | quiz_attempts | id, user_id, mode, score, duration | 刷题记录 |
| QuizAnswer | quiz_answers | id, attempt_id, quiz_id, user_answer, is_correct | 作答记录 |
| WrongQuestion | wrong_questions | id, user_id, quiz_id, wrong_count, is_mastered, next_review | 错题本 |
| Example | examples | id, chapter, title, question, steps_json | 例题 |

## 八、数据库迁移

系统内置自动迁移机制，在应用启动时自动执行：
- 新增表的创建
- 已有表的字段补充
- 废弃字段的移除（如 quizzes 表的 confidence、match_status 字段）

迁移函数位于 [app/__init__.py](file:///d:/唐宏/实验室/离散数学智能体/app/__init__.py)，包括：
- `_migrate_quizzes()` - 题库迁移
- `_migrate_users()` - 用户表迁移
- `_migrate_wrong_book_tables()` - 错题本迁移
- `_migrate_courseware_mode()` - 课件模式字段迁移

## 八、安全性

| 安全措施 | 说明 |
|----------|------|
| 密码加密 | 使用 Werkzeug 的 generate_password_hash 加密存储 |
| 会话管理 | Flask session 管理用户登录状态 |
| 权限控制 | @login_required 和 @admin_required 装饰器 |
| 输入验证 | 前端表单验证 + 后端参数校验 |
| XSS 防护 | Jinja2 自动 HTML 转义 |
| 文件上传 | 白名单扩展名检查 |

## 九、总结

离散数学智能教学体是一个功能完整、设计精美的智能化教学辅助平台，涵盖了离散数学学习的全流程：
